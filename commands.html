Killua: Commands




Commands



Command Groups


Killua has Moderation, Fun, Economy, and Miscellaneous commands.



Moderation Commands


k!prefix [string] - Sets a new prefix for Killua for the server, can only be used by server admins.
k!ban [user] [optional reason] - Bans a member and deletes their messages from over the last 24 hours.
k!unban [user or name and tag] - Unbans the specified user.
k!mute [user] [optional time in minutes] [optional reason] - Mutes a user. If a time is not specified, the user will be muted for an unlimited amount of time.
k!unmute [user] [optional reason] - Unmutes the specified user.



Fun Commands


k!quote [user] [message] - Send a screenshot of a user saying what you defined. Use -c at the start of text for compact mode or -l for light mode or both.
k!cmm [text] - Sends the Change My Mind meme with the text you defined.
k!hug [user] - Hugs the defined user.
k!topic - Get a topic with this command!
k!calc [math expression] - Calculates the given expression.
k!translate [source language or auto] [target language] [text] - Translates the given text to the target language.
k!8ball [question] - Will answer the provided question.
k!avatar [user] - Sends the avatar picture of the specified user.
k!book [book name] - Get info about the provided book.
k!emojaic [user, id, or link to image] - Turns the provided image into emojis!
k!image [title] - Gives you the best DuckDuckGo image for your title.
k!f [type] [user, id, or link to image] - Applies the specified filter to the image. To get the list of filters, just type k!f.



Economy Commands


k!daily - Claim your daily points!
k!rps [user] [optional bet] - Play Rock, paper, scissors with another user! You can bet points on the game.
k!profile [user id or mention] - View the profile of the defined user.
k!give [user] [amount] - Gives the defined user the specified amount of your coins.
k!server - Gives you info about the current server.

Miscellaneous Commands


k!info - Gives you information about Killua!
k!patreon - Gives you a link to the Killua Patreon.
k!invite - Gives you the invite link.
k!codeinfo [command] - Gives you insights on a command.
k!permissions - Will list the permissions Killua has on the server.
k!bug [command name or other] [description] - You can report bugs with this command. Abusing it will get you blacklisted.
k!fp [type of feedback] [feedback] - Send feedback directly to Kile! Abuse or spam of this command will result in blacklisting.
