Moderation Commands
Command Description Format
prefix Sets a new prefix for Killua for the server, can only be used by server admins. k!prefix [string]
ban Bans a member and deletes their messages from over the last 24 hours. k!ban [user] [optional reason]
kick Kicks the member specified. k!kick [user] [optional reason]
unban Unbans the the user. Please specify the ID or the name and tag. k!unban [userId or Nameandtag]
mute Mutes a user for the given amount of time you specify, or unlimited. k!mute [user] [optional time] [optional reason]
unmute Unmutes the user. k!unmute [user] [optional reason]


Fun Commands
Command Description Format
quote Send a screenshot of a user saying what you defined. Use -c at the start of text for compact mode or -l for light mode or both. k!quote [user] [text]
cmm Sends the change my mind meme with the text you defined. k!cmm [text]
hug Hugs the user specified. k!hug [user]
pat Pats the specified user, similar to the hug command. k!pat [user]
topic Get a topic to talk about with this command. k!topic
calc Calculates the specified mathematical expression. k!calc
translate Translates the text to the given target language. k!translate [source language or auto] [target language] [text]
8ball Killua will answer the question. k!8ball [question]
avatar Retrieves the avatar of the user. k!avatar [user]
book Gets information about the book. k!book [book title]
emojaic Changes the provided image to emojis (try this!) k!emojaic [user, id, or link]
image Gives you the best DuckDuckGo images for the title. k!image [title]
f To get a list of supported types, enter the command without arguments. k!f [type] [user, id, or link]


Economy Commands
Command Description Format
daily Gives you your daily Jenny. k!daily
rps Challenges someone to a game of rock paper scissors. You can make bets. k!rps [user] [optional bet]
profile Shows info on a specified user. k!profile [user]
give Gives another user the specified amount of Jenny. k!give [user] [amount]
server Gives you information about the current server. k!server
bal Shows the balance of the specified user. k!bal [optional user]


Todo Commands
Command Description Format
create Creates a todo-list with an interactive and easy setup. k!todo create
edit Allows you to edit a list and enables the use of lots of commands. Requires being owner or having edit permissions. k!todo edit [list id]
view When in edit mode, will show the entries of the current list you are editing. When not in edit mode, it shows the todo list with the specified ID (must have sufficient permissions). k!todo view [list id]
info Gives insights about the todo task specified or the current list if no argument is provided, when in edit mode. When not in edit mode, it gives information on the list or a task, if specified (requires sufficient permissions). k!todo info [todo task number (optional) or todo list ID] [optional todo task number]
invite Sends the specified user a DM if possible and if they accept the invitation, they get the specified permission for that todo list. Only available in editor mode. Available types: editor, viewer. k!todo invite [mention or user id] [type]
kick Only available in editor mode. Requires list owner permissions. Takes every permission the specified user had in the current list. k!todo kick [mention or user id]
buy You are able to purchase extra space, a custom color or a custom thumbnail for your list with this command. Only available in editor mode. Available types: thumbnail, color, space. k!todo buy [type]
shop Displays a menu of how much what you can buy is. k!todo shop
exit Exits the current list you are editing. Only available in editor mode. k!todo exit
status Only available in editor mode. Enables you to change the todo-list's status you previously specified in todo create. k!todo status
name Only available in editor mode. Enables you to change the todo-lists's name you previously specified in todo create. k!todo name
autodelete Only available in editor mode. Enables you to change the todo-list's setting if it should auto-delete a todo that has been marked as "done" you previously specified in todo-create. k!todo autodelete
color Only available in editor mode. Enables you to change the todo-list's color if you have purchased it with todo buy color. k!todo color
thumbnail Only available in editor mode. Enables you to change the todo list's color if you have purchased it with todo buy thumbnail. k!todo thumbnail
custom_id Only available in editor mode. Enables you to change the todo-list's ID if you have it set in todo create (only possible if you are premium supporter). k!todo custom_id
remove Only available in editor mode. Removes the todo task with the specified number. k!todo remove [todo-task-number]
add Only available in editor mode. Adds "todo" to your todos of your current list. k!todo add [todo]
mark Only avialable in editor mode. Marks the todo task specified with the comment specified. If the comment is "done" and you specified that a task should be deleted when it’s done, this will delete the task. If the comment is "-rm" the comment to that todo task is entirely removed. k!todo mark [todo-task-number] [marked-as]
assign Only available in editor mode. Required permissions of user you assign the tast to: editor or owner. Assigns a todo task to a team member who will be informed through dms. A todo task can have multiple people assigned to it. If you decide to remove a user from a task, use the command as normal and add "-rm" at the end. k!todo assign [todo task number] [mention/user id] [optional-rm]
delete Only available in editor mode. Requires owner permissions of the current list. Deletes the current list you are on. k!todo delete
lists Gives you a list of what you own, view, and edit so you don't have to memorize the IDs. k!todo lists


Other Commands
Command Description Format
