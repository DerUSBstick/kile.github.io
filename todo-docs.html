Killua: Todo Commands




Commands




k!todo edit [list id]

Required: being owner or having edit permissions to the list

Allows you to edit a list and enables the use of lots of commands.



k!todo view [todo-list-id]

In edit mode:
Shows the entries of the current list.
Not in edit mode:
Required: Being the owner or having viewer or edit permissions to the list
Shows the todo list with the specified id



k!todo info [todo-task-number-optional/todo-list-id] [optional-todo-task-number]

In edit mode:
Gives insights about the todo task specified or the current list if no argument is provided
Not in edit mode:
Required: being owner or having viewer or edit permissions to the list
Gives info about either the list or one of it's todo tasks if you specify it



k!todo invite [mention/user-id] [type]

Only available in editor mode
Available types: editor, viewer
Sends the specified user a DM if possible and if they accept the invitation, they get the specified permission for that todo list



k!todo kick [mention/user-id]

Only available in editor mode
Requires list owner permissions
Takes every permission that user had in the current list



k!todo buy [type]

Only available in editor mode
Available types: thumbnail, color, space
You are able to purchase extra space, a custom color or a custom thumbnail for your list with this command



k!todo shop


Displays a menu of how much what you can buy is



k!todo exit

Only available in editor mode
Exits the current list you are editing



k!todo status


Only available in editor mode
Enables you to change the todo-list's status you previously specified in todo create



k!todo name

Only available in editor mode
Enables you to change the todo-list's name you previously specified in todo create



k!todo autodelete


Only available in editor mode
Enables you to change the todo-list's setting if it should auto-delete a todo that has been marked as "done" you previously specified in todo create



k!todo color

Only available in editor mode
Enables you to change the todo-list's color if you have purchased it with todo buy color



k!todo thumbnail

Only available in editor mode
Enables you to change the todo list's color if you have purchased it with todo buy thumbnail



k!todo custom_id

Only available in editor mode
Enables you to change the todo-list's color if you have set it in todo create (only possible if you are a premium supporter)



k!todo remove [todo-task-number]

Only available in editor mode
Removes the todo task with the specified number



k!todo add [todo]

Only available in editor mode
Adds "todo" to your todos of your current list



k!todo mark [todo-task-number] [marked-as]

Only available in editor mode
Marks the todo task specified with the comment specified. If the comment is "done" and you specified that a task should be deleted when itâ€™s done, this will delete the task. If the comment is "-rm" the comment to that todo task is entirely removed



k!todo assign [todo-task-number] [mention/user-id] [optional-rm]

Only available in editor mode
Required permissions of user you assign the task to: editor or owner
Assigns a todo task to a team member who will be informed through dms. A todo task can have multiple people assigned to it. If you decide to remove a user from a task, use the command as normal and add "-rm" at the end



k!todo delete

Only available in editor mode
Requires owner permissions of the current list
Deletes the current list you are on



k!todo lists

Gives you a list of what lists you own, can view and edit so you don't have to memorize the IDs
